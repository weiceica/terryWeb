Travelour

Travelour is a travel planner that has a built in chatbot built with the RASA intent classification model responding to user input.

The app was built with Next.js, while the NLP model was trained with over 8000 data points. Data is hosted on GCP Firebase, while NextAuth.js is used for auth.
