Projects

TSP Visualizer

The TSP Visualizer is a web-based application that I created to help visualize various Travelling Salesman problem algorithms, heurestics, and many other nodal visualizations. It is coded with front-end web development tools from CSS and HTML, with the algorithms and data storage being coded in javascript. The next steps of this project is to remodel the page in React framework and take the algorithms to the backend through creating a restful API.
https://weiceica.github.io/TSP/

World Cup Star

The World Cup Star project is a machine-learning/software engineering project to help use data that is availible from the internet to predict the results of the matches in the World Cup using machine learning models in tensorflow. The next steps of this project is to create live updates through webscraping online databases and storing it in hosted database on Microsoft Azure. When the Qatar 2022 world cup begins, live updates to the machine learning model would be incorporated into the calculations of the predictions that it makes.
